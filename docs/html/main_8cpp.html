Main function to initialize and run a robot swarm control application.

This function initializes ROS 2, creates a multi-threaded executor, generates a specified number of robot nodes with random parameters, and adds them to the executor. It also creates a SwarmMaster node, adds it to the executor, and starts spinning the executor.
